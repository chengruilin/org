*** Primitive types
    Each object blongs to one and only one primitive type.
    - integer
    - float
    - cons
    - symbol
    - string
    - vector
    - hash-table
    - subr
    - byte-code function
    - record
    - buffer

    Such type declarations do not exist in Emacs Lisp.
